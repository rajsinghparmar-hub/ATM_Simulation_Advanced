# ATM Simulation (Advanced)

## Overview
This is a Python-based ATM simulation program that allows a user to perform banking operations such as checking balance, depositing money, withdrawing money, changing PIN, and viewing transaction history.

## Features

1. PIN Verification  
   - Default PIN: 1234  
   - Maximum 3 attempts to enter correct PIN

2. Check Balance  
   - View current account balance

3. Deposit Money  
   - Add money to the account  
   - Transaction history updated with deposits

4. Withdraw Money  
   - Withdraw money respecting daily withdrawal limit  
   - Transaction history updated with withdrawals

5. Change PIN
   - Change the ATM PIN securely

6. Transaction History  
   - Shows all deposits and withdrawals

7. Exit  
   - Exit the program gracefully
   ```bash
   python atm_simulation.py
